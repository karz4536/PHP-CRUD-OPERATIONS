CONNECT TO DATABASE:
PHP mysqli_connect() function is used to connect with MySQL database. It returns resource if connection is established or null. resource mysqli_connect (server, username, password)

CLOSING THE CONNECTION:
PHP mysqli_close() function is used to disconnect with MySQL database. It returns true if connection is closed or false. bool mysqli_close(resource $resource_link)
